Fundamentals of Sheel Scripting
  1. ls -list the files and folders
  2. pwd - present working directory
  3. cd directoryname/ - change directory
  4. cd .. - go back to previous directory
  5. cd ../.. - go back to 2 previous directory
  6. ls -ltr - we will get details of the files and directory with owner, timestaps
  7. touch mreddy - Create a file
  8. vi test - Create and write in the file esc i esc :wq
  9. cat test -to print the file
  10. mkdir mreddy1 - create a directory
  11. rm -r mreddy1 - Remove a directory
  12. top - used to get all the details like memory, diskspace, processors
  13. free -g - used to get the memory of the CPU -*
  14. nproc - number of processors -*
  15. df -f - disk -*
  16. ctrl+c ->to exit the running command
